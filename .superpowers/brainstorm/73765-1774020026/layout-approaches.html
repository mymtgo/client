Dashboard Layout Approaches

Three ways to organize the redesigned dashboard. Click your preference.

A

Hero + Grid — "Momentum Dashboard"

A bold top section with your current form front and center — streak indicator, rolling winrate gauge, and active league. Below that, a 2-column grid of widget cards: format chart, matchup spread, play/draw split, session recap. Dense but scannable.

┌─────────────────────────────────────────┐
│ 🔥 5W streak │ 62% last 20 │ League 3-1 │
│         [ timeframe selector ]          │
├────────────────────┬────────────────────┤
│ Match/Game KPIs │ Play/Draw Split │
├────────────────────┼────────────────────┤
│ Format Chart │ Matchup Spread │
├────────────────────┼────────────────────┤
│ Session Recap │ Best/Worst Decks │
└────────────────────┴────────────────────┘
B

Stacked Narrative — "Story of Your Session"

Full-width sections that tell a story top to bottom: "Here's your momentum" → "Here's what you played" → "Here's what you faced" → "Here's where to improve". Each section is a focused card. Reads like a report. Simpler layout, more breathing room.

┌─────────────────────────────────────────┐
│ MOMENTUM: streak, rolling %, trend line │
├─────────────────────────────────────────┤
│ SESSION: last session matches + result │
├─────────────────────────────────────────┤
│ META: top archetypes faced + your W/L │
├─────────────────────────────────────────┤
│ PERFORMANCE: format chart, play/draw │
├─────────────────────────────────────────┤
│ DECKS: best/worst + league status │
└─────────────────────────────────────────┘
C

Hybrid — "Command Center"

Compact KPI strip at top (streak, winrates, league). Then a mixed grid: wider cards for charts/tables, narrower cards for stats. Maximizes information density without being overwhelming. The grid adapts — important widgets are larger.

┌──────────┬──────────┬──────────┬─────────┐
│ 🔥 5W │ 62% M.WR │ 58% G.WR │ Lg: 3-1 │
├──────────┴──────────┴──────────┴─────────┤
│ Format Trend Chart (wide)                │
├───────────────────────┬───────────────────┤
│ Session Recap │ Matchup Spread │
│ (last 5 matches) │ (top 5 archetypes)│
├───────────────────────┼───────────────────┤
│ Play/Draw Split │ Best/Worst Deck │
└───────────────────────┴───────────────────┘

What's the same across all three:

  • Timeframe selector stays
  • Active league widget stays
  • Recent matches table removed (lives on /matches page)
  • New widgets: streak indicator, rolling form, session recap, matchup spread, play/draw split
  • Existing widgets improved: KPIs, format chart, best/worst deck
